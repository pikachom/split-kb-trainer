# ⌨️ Split KB Trainer

Corne 38-key (3×6 column-stagger) 스플릿 키보드 타자연습 웹앱

## 실행법

```bash
# 그냥 브라우저로 열기 (인터넷 불필요)
open index.html

# 또는 로컬 서버로 실행
npx serve .
python3 -m http.server 8080
```

## 기능

- **5가지 연습 모드**: 영문 / 한글 / 특수문자 / 혼합 / 레이어 연습
- **키보드 시각화**: Base / Lower / Raise 레이어 전환, 다음 입력할 키 하이라이트
- **손가락 컬러 가이드**: 검지/중지/약지/소지/엄지별 색상 표시
- **실시간 통계**: WPM, 정확도, 경과시간, 오류수
- **오류 히트맵**: 자주 틀리는 키 시각화
- 한글 IME 지원 (조합 중 오류 없음)

## 키맵 가정 (Corne 38-key)

| 레이어 | 내용 |
|-------|------|
| Base | QWERTY, 엄지: DEL / SPC / LWR (왼), RSE / ENT / BSP (오른) |
| Lower (왼엄지 홀드) | 숫자 1~0, 특수문자 !@#$%^&*() |
| Raise (오른엄지 홀드) | 방향키, F1~F12, PgUp/PgDn |

## 컬럼 스태거 적응 팁

1. 손을 **수직으로 내려찍듯** 치면 훨씬 자연스러워요
2. 홈 포지션(ASDF / JKL;)을 항상 유지하세요
3. 엄지로 Space와 레이어 키를 번갈아 쓰는 데 집중
4. 처음엔 WPM보다 **정확도 우선**으로 연습
